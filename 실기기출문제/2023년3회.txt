1. 다음은 Java 코드이다. 올바른 출력 결과를 작성하시오.

package dummy;

class A {
    // 초기는 부모로 오고 override 체크
    public void paint() {
        System.out.print("A");
        draw();
    }
    public void draw() {
        System.out.print("B");
        // override 
        draw();
    }
}
 
class B extends A {
    // 자식에서 override 되있으니 이거 실행
    public void paint() {
        // 부모꺼 강제호출
        super.draw();
        System.out.print("C");
        // 자기꺼 호출
        this.draw();
    }
    public void draw() {
        System.out.print("D");
    }
}
public class Test11 {	
	public static void main(String[] args) {
        // upcasting. 포커스는 A로 맞춤
		A b = new B();
        b.paint();
        b.draw();
	}
}

7. 다음 설명은 서버 접근 통제의 유형이다. 괄호 안에 들어갈 용어를 작성하시오. (영어 약자로 작성하시오.)
MAC
 - 군대 같이 계급으로 굴러가고 딱딱함
 - 규칙 기반(Rule-Based) 접근통제 정책으로, 객체에 포함된 정보의 허용 등급과 접근 정보에 대하여 주체가 갖는 접근 허가 권한에 근거하여 객체에 대한 접근을 제한하는 방법이다.

RBAC
 - 역할(Role)

DAC
 - 소유자, owner
 - 신분 기반(Identity-Based) 접근통제 정책으로, 주체나 그룹의 신분에 근거하여 객체에 대한 접근을 제한하는 방법이다.
