* SQL
 - DB 조작하는 명령어. 영어의 자연어를 닮음
 - DCL: 권한, DB 조작관련
   DDL: 테이블 생성, 수정, 삭제, 변경
   DML: 행(데이터) 조회, 추가, 수정, 삭제

* 관계대수
 - 데이터 가져오는 방식을 수학기호로 표현
 - 절차적
 - 일반집합 연산자 : 합집합(∪), 교집합(∩), 차집합(-), 교차곱(x)
    테이블과 테이블을 집합처럼 계산한다.
 - 순수관계 연산자 : SELECT(σ), PROJECT(π), JOIN(⋈), DIVISION(÷)
    테이블에서 원하는 행/열/관계를 뽑거나 연결한다.
* 관계해석
 - 원하는 데이터가 만족해야 하는 조건을 논리식으로 표현하는 
   **비절차적 질의 언어**이다
