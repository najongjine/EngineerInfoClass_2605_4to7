1. C언어 기본 구조 복습
==================================================

C언어 코드는 보통 아래 모양으로 시작한다.

```c
#include <stdio.h>

int main() {
    printf("Hello C\n");
    return 0;
}
```

처음에는 모든 줄을 완벽하게 이해하지 않아도 된다.
지금 단계에서는 다음 정도로 이해하면 충분하다.

- #include <stdio.h>
  - printf를 쓰기 위해 필요한 준비 코드

- int main()
  - 프로그램이 시작되는 위치

- { }
  - main 함수의 시작과 끝

- printf(...)
  - 화면에 글자나 값을 출력하는 명령

- return 0;
  - 프로그램을 정상적으로 끝낸다는 의미

중요한 습관이 하나 있다.

코드를 볼 때는 위에서 아래로 읽는다.
컴퓨터도 특별한 지시가 없으면 위에서 아래로 한 줄씩 실행한다.

==================================================
2. 변수란 무엇인가
==================================================

변수는 값을 저장하는 이름 붙은 공간이다.

비전공자에게는 "상자"라고 생각하면 쉽다.

예를 들어 다음 코드를 보자.

```c
int age = 20;
```

이 코드는 다음 뜻이다.

- int
  - 정수를 저장하겠다는 뜻

- age
  - 변수 이름

- =
  - 오른쪽 값을 왼쪽 변수에 넣는다는 뜻

- 20
  - 변수에 넣을 값

즉, "age라는 상자에 20을 넣는다"라고 읽으면 된다.

주의할 점:
C언어에서 = 는 수학의 "같다"가 아니다.
C언어에서 = 는 "오른쪽 값을 왼쪽에 넣어라"라는 뜻이다.

```c
age = 21;
```

이 코드는 "age는 21과 같다"가 아니라,
"age라는 변수에 21을 새로 넣어라"라는 뜻이다.

==================================================
3. 자료형: int, double, char
==================================================

자료형은 변수에 어떤 종류의 값을 넣을지 정하는 것이다.

오늘은 세 가지만 익힌다.

--------------------------------------------------
1) int
--------------------------------------------------

int는 정수를 저장한다.

정수는 소수점이 없는 숫자이다.

예:
- 10
- -3
- 0
- 2026

```c
int score = 80;
```

--------------------------------------------------
2) double
--------------------------------------------------

double은 실수를 저장한다.

실수는 소수점이 있는 숫자이다.

예:
- 3.14
- 72.5
- -1.25

```c
double height = 172.5;
```

--------------------------------------------------
3) char
--------------------------------------------------

char는 문자 하나를 저장한다.

문자 하나는 작은따옴표로 감싼다.

```c
char grade = 'A';
```

주의:

```c
char grade = 'A';   // 맞음
char grade = "A";   // 오늘 단계에서는 틀렸다고 생각하기
```

작은따옴표 'A'는 문자 하나이다.
큰따옴표 "A"는 문자열이다.
문자열은 나중에 배열을 배울 때 다시 다룬다.

==================================================
4. printf로 변수 출력하기
==================================================

printf는 화면에 값을 출력한다.

변수를 출력하려면 형식 지정자를 사용한다.

- int 출력: %d
- double 출력: %f
- char 출력: %c

예제 1:

```c
#include <stdio.h>

int main() {
    int age = 20;
    double height = 172.5;
    char grade = 'A';

    printf("age = %d\n", age);
    printf("height = %f\n", height);
    printf("grade = %c\n", grade);

    return 0;
}
```

예상 출력:

```text
age = 20
height = 172.500000
grade = A
```

double을 %f로 출력하면 기본적으로 소수점 아래 6자리까지 출력된다.
소수점 한 자리만 보고 싶으면 %.1f를 사용한다.

```c
printf("height = %.1f\n", height);
```

출력:

```text
height = 172.5
```

==================================================
6. 변수값은 바뀔 수 있다
==================================================

변수는 한 번 값을 넣고 끝나는 것이 아니다.
중간에 다른 값을 다시 넣을 수 있다.

예제 2:

```c
#include <stdio.h>

int main() {
    int score = 10;

    printf("1: score = %d\n", score);

    score = 20;
    printf("2: score = %d\n", score);

    score = 30;
    printf("3: score = %d\n", score);

    return 0;
}
```

출력:

```text
1: score = 10
2: score = 20
3: score = 30
```

변수값 변화표:

| 실행 위치 | 실행 코드 | score 값 |
|---|---|---|
| 처음 | int score = 10; | 10 |
| 1번 출력 | printf(...) | 10 |
| 값 변경 | score = 20; | 20 |
| 2번 출력 | printf(...) | 20 |
| 값 변경 | score = 30; | 30 |
| 3번 출력 | printf(...) | 30 |

코드 추적 문제를 풀 때는 이런 표를 손으로 쓰면 실수가 줄어든다.

==================================================
7. 로그 찍기란 무엇인가
==================================================

로그 찍기는 중간값을 확인하기 위해 printf를 추가하는 것이다.

시험장에서는 실행할 수 없지만,
공부할 때는 반드시 로그를 찍어봐야 한다.
그래야 나중에 눈으로만 봐도 흐름이 보인다.

예를 들어 아래 코드는 마지막 값만 출력한다.

```c
#include <stdio.h>

int main() {
    int a = 5;
    int b = 3;

    a = a + b;
    b = a - b;

    printf("%d %d\n", a, b);

    return 0;
}
```

처음 보면 a와 b가 헷갈릴 수 있다.
그래서 중간 로그를 추가한다.

```c
#include <stdio.h>

int main() {
    int a = 5;
    int b = 3;

    printf("처음: a = %d, b = %d\n", a, b);

    a = a + b;
    printf("a = a + b 이후: a = %d, b = %d\n", a, b);

    b = a - b;
    printf("b = a - b 이후: a = %d, b = %d\n", a, b);

    printf("마지막: %d %d\n", a, b);

    return 0;
}
```

출력 흐름:

```text
처음: a = 5, b = 3
a = a + b 이후: a = 8, b = 3
b = a - b 이후: a = 8, b = 5
마지막: 8 5
```

핵심:

```c
a = a + b;
```

이 코드는 오른쪽을 먼저 계산한다.
현재 a는 5, b는 3이므로 오른쪽 a + b는 8이다.
그 다음 8을 왼쪽 a에 넣는다.
그래서 a가 8로 바뀐다.

==================================================
8. 산술연산자
==================================================

C언어의 기본 산술연산자는 다음과 같다.

| 연산자 | 의미 | 예 |
|---|---|---|
| + | 더하기 | 10 + 3 |
| - | 빼기 | 10 - 3 |
| * | 곱하기 | 10 * 3 |
| / | 나누기 | 10 / 3 |
| % | 나머지 | 10 % 3 |

예제 3:

```c
#include <stdio.h>

int main() {
    int a = 10;
    int b = 3;

    printf("a + b = %d\n", a + b);
    printf("a - b = %d\n", a - b);
    printf("a * b = %d\n", a * b);
    printf("a / b = %d\n", a / b);
    printf("a %% b = %d\n", a % b);

    return 0;
}
```

예상 출력:

```text
a + b = 13
a - b = 7
a * b = 30
a / b = 3
a % b = 1
```

주의:

```c
int a = 10;
int b = 3;
printf("%d\n", a / b);
```

결과는 3이다.
3.333333이 아니다.

왜냐하면 a와 b가 둘 다 int이기 때문이다.
정수끼리 나누면 소수점 아래를 버린 정수 몫이 나온다.

10을 3으로 나누면

- 몫: 3
- 나머지: 1

그래서

```c
10 / 3 == 3
10 % 3 == 1
```

9. % 연산자 자세히 보기
==================================================

%는 나머지를 구하는 연산자이다.

처음에는 쓸모없어 보이지만 시험에 자주 등장한다.

예:

```c
printf("%d\n", 7 % 2);
```

7을 2로 나누면

- 몫: 3
- 나머지: 1

출력은 1이다.

자주 나오는 패턴:

```c
number % 2
```

이 값이 0이면 짝수이다.
이 값이 1이면 홀수이다.

예:

```c
8 % 2 == 0
9 % 2 == 1
```

오늘은 조건문을 깊게 다루지 않지만,
나중에 if문에서 짝수/홀수 판별에 자주 사용한다.

11. 대입연산자와 누적 계산
==================================================

정보처리기사 실기 C언어 문제에서 자주 헷갈리는 코드가 있다.

```c
sum = sum + 5;
```

수학처럼 보면 이상하다.
"sum이 sum + 5와 같다"는 말은 말이 안 된다.

하지만 C언어에서는 = 가 대입이다.
오른쪽을 먼저 계산하고, 그 결과를 왼쪽 변수에 넣는다.

예제 4:

```c
#include <stdio.h>

int main() {
    int sum = 0;

    sum = sum + 5;
    printf("1: sum = %d\n", sum);

    sum = sum + 3;
    printf("2: sum = %d\n", sum);

    sum = sum + 10;
    printf("3: sum = %d\n", sum);

    return 0;
}
```

변수값 변화표:

| 실행 코드 | 오른쪽 계산 | sum에 저장되는 값 |
|---|---|---|
| int sum = 0; | 0 | 0 |
| sum = sum + 5; | 0 + 5 | 5 |
| sum = sum + 3; | 5 + 3 | 8 |
| sum = sum + 10; | 8 + 10 | 18 |

출력:

```text
1: sum = 5
2: sum = 8
3: sum = 18
```

이런 식으로 기존 값에 무언가를 계속 더하는 것을 "누적"이라고 한다.
나중에 반복문을 배울 때 매우 중요하다.

==================================================
12. 복합 대입연산자
==================================================

아래 두 코드는 같은 의미이다.

```c
sum = sum + 5;
sum += 5;
```

둘 다 sum에 5를 더해서 다시 sum에 저장한다.

자주 쓰는 형태:

| 긴 표현 | 짧은 표현 | 의미 |
|---|---|---|
| a = a + 3; | a += 3; | a에 3을 더한다 |
| a = a - 3; | a -= 3; | a에서 3을 뺀다 |
| a = a * 3; | a *= 3; | a에 3을 곱한다 |
| a = a / 3; | a /= 3; | a를 3으로 나눈 몫을 저장한다 |
| a = a % 3; | a %= 3; | a를 3으로 나눈 나머지를 저장한다 |

예제 5:

```c
#include <stdio.h>

int main() {
    int a = 10;

    a += 5;
    printf("1: %d\n", a);

    a *= 2;
    printf("2: %d\n", a);

    a -= 4;
    printf("3: %d\n", a);

    a /= 3;
    printf("4: %d\n", a);

    return 0;
}
```

손으로 추적:

| 실행 코드 | a 값 |
|---|---|
| int a = 10; | 10 |
| a += 5; | 15 |
| a *= 2; | 30 |
| a -= 4; | 26 |
| a /= 3; | 8 |

마지막 a /= 3에서 26 / 3은 8이다.
정수 나눗셈이므로 소수점 아래는 버린다.

출력:

```text
1: 15
2: 30
3: 26
4: 8
```

13. 증감연산자 ++, --
==================================================

증감연산자는 변수값을 1 증가시키거나 1 감소시킨다.

```c
a++;
```

위 코드는 아래와 같은 뜻이다.

```c
a = a + 1;
```

반대로

```c
a--;
```

위 코드는 아래와 같은 뜻이다.

```c
a = a - 1;
```

예제 6:

```c
#include <stdio.h>

int main() {
    int a = 5;

    printf("처음: %d\n", a);

    a++;
    printf("a++ 이후: %d\n", a);

    a++;
    printf("a++ 이후: %d\n", a);

    a--;
    printf("a-- 이후: %d\n", a);

    return 0;
}
```

출력:

```text
처음: 5
a++ 이후: 6
a++ 이후: 7
a-- 이후: 6
```

오늘은 우선 이렇게 이해한다.

- a++ : a를 1 증가
- a-- : a를 1 감소

나중에는 ++a와 a++의 차이도 배운다.
오늘은 printf 안에 a++를 직접 넣는 복잡한 형태는 깊게 다루지 않는다.

14. 종합 예제: 변수, 연산자, 로그 찍기

아래 코드는 오늘 배운 내용을 모두 섞은 예제이다.

```c
#include <stdio.h>

int main() {
    int a = 7;
    int b = 3;
    int result = 0;

    printf("처음: a = %d, b = %d, result = %d\n", a, b, result);

    result = a + b;

}
```

19. 실습 과제 E: 로그를 직접 추가하기
==================================================

아래 원본 코드는 마지막 결과만 출력한다.

```c
#include <stdio.h>

int main() {
    int a = 9;
    int b = 4;
    int c = 0;

    c = a / b;
    a = a % b;
    b = b + c;

    printf("%d %d %d\n", a, b, c);

    return 0;
}
```

해야 할 일:

1. 실행 전에 마지막 출력 결과를 예상한다.
2. 각 계산 줄 아래에 printf 로그를 추가한다.
3. 실행 결과가 예상과 같은지 확인한다.

로그 추가 예시:

```c
#include <stdio.h>

int main() {
    int a = 9;
    int b = 4;
    int c = 0;

    printf("처음: a = %d, b = %d, c = %d\n", a, b, c);

    c = a / b;
    printf("c = a / b 이후: a = %d, b = %d, c = %d\n", a, b, c);

    a = a % b;
    printf("a = a %% b 이후: a = %d, b = %d, c = %d\n", a, b, c);

    b = b + c;
    printf("b = b + c 이후: a = %d, b = %d, c = %d\n", a, b, c);

    printf("마지막: %d %d %d\n", a, b, c);

    return 0;
}
```

정답 추적:

| 실행 코드 | a | b | c |
|---|---:|---:|---:|
| 초기값 | 9 | 4 | 0 |
| c = a / b; | 9 | 4 | 2 |
| a = a % b; | 1 | 4 | 2 |
| b = b + c; | 1 | 6 | 2 |

최종 출력:

```text
1 6 2
```

주의:
printf에서 % 글자 자체를 출력하고 싶으면 %%라고 적는다.

```c
printf("a %% b\n");
```

출력:

```text
a % b
```

20. 오늘의 핵심 정리
==================================================

1. 변수는 값을 저장하는 이름 붙은 공간이다.

```c
int age = 20;
```

2. 자료형은 값의 종류를 정한다.

```c
int number = 10;
double rate = 3.14;
char grade = 'A';
```

3. printf는 값을 출력한다.

```c
printf("%d\n", number);
printf("%.2f\n", rate);
printf("%c\n", grade);
```

4. = 는 같다라는 뜻이 아니라 대입이다.

```c
a = a + 3;
```

오른쪽 a + 3을 먼저 계산하고, 그 결과를 왼쪽 a에 넣는다.

5. 정수끼리 나누면 정수 몫이 나온다.

```c
10 / 3  // 3
10 % 3  // 1
```

6. ++와 --는 1씩 증가하거나 감소한다.

```c
a++;
a--;
```

7. 코드 추적은 표로 한다.

| 줄 | a | b | result |
|---|---:|---:|---:|
| 초기값 |  |  |  |
| 1번째 계산 |  |  |  |
| 2번째 계산 |  |  |  |

8. 공부할 때는 printf 로그를 찍는다.

```c
printf("a = %d, b = %d\n", a, b);
```

로그를 많이 찍어본 사람이 나중에 로그 없이도 코드를 잘 읽는다.
